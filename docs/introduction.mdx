# What is this wiki for? 
I created this wiki to: 
- Help people with basic problems they have with the pack
- Explain how to download and use pre-release versions of the modpack from Github
- Explain some of the optimizations I have used in the pack (for other modpack developers to use and for me to look back at later)
- Explain what custom things I have done for the pack and why

Note: I probably won't update this documentation until I release Create+ v6 or it is very close to full release.
